# Village-Environmental-Awareness-and-Risk-Detection-System
Village environmental awareness and risk detection system using C++ and OpenGL/GLUT. Simulates a rural village with rain, stagnant water, mosquito activity, day-night mode, and dengue risk alerts to promote public health awareness.

## Project Overview
The **Village Environmental Awareness and Risk Detection System** is a computer graphics project developed using **C++** and **OpenGL/GLUT**. The project simulates a rural village environment and visualizes environmental health risks caused by rainfall, stagnant water, and mosquito activity.

The main goal of this project is to create awareness about how environmental conditions in village areas can lead to public health problems such as **dengue risk**. The system combines visual simulation with interactive controls to demonstrate changing village conditions in real time.

## Features
- Day and night mode
- Rain animation
- Water level increase and decrease
- Pond / stagnant water visualization
- Mosquito movement animation
- Dengue risk warning display
- Interactive keyboard controls

## How to Run
### Using MinGW-w64
Compile the project with:

```bash
g++ main.cpp -o app.exe -lfreeglut -lopengl32 -lglu32

Then run:

./app.exe
Controls

d → Day mode

n → Night mode

r → Toggle rain on/off

w → Increase water level

s → Decrease water level

Project Purpose

This project is designed to raise awareness about environmental risk factors in village areas. It shows how stagnant water and rainfall can create mosquito breeding conditions and increase dengue risk. The project demonstrates how computer graphics can be used to represent real-life awareness and safety scenarios.
